# Showreel 风格 HTML Slide Editor — 界面目标文档

> 创建日期：2026-06-30  
> 当前版本：v0.7.0（P3 完成）  
> 参考产品：Showreel / Gamma / Tome

---

## 1. 产品定位

本项目**不是**传统 PowerPoint 克隆。它是一个**模板驱动的现代 HTML Slide Editor**：

- 左侧页面列表
- 中间 16:9 演示画布
- 右侧内容/模板/设置 inspector
- 顶部提供编辑、演示、导出、播放操作

用户通过选择模板、填写结构化内容表单来快速制作口播视频配套演示画面、产品演示页、汇报演示页，而非通过自由画布拖拽来拼凑幻灯片。

### 2.1 左侧页面栏

- 页面列表，纵向排列
- 每项显示：
  - 页码（01, 02, 03 ...）
  - 模板类型标签（FLOW / STACK / CODE / STAT / QUOTE / STATEMENT / FREE）
  - 页面标题摘要（截取标题文字第一行）
- 当前页高亮（浅色背景 + 左侧强调色条）
- 底部"新增页面"按钮
- 每项悬停显示：删除、复制、上下移动入口（icon buttons）

### 2.2 中间画布

- 画布背景：浅色点阵网格（dot-grid pattern）
- 16:9 幻灯片卡片：白色、圆角 12px、柔和阴影
- 画布顶部信息栏：页码、比例（16:9）、分辨率（1280×720）
- 内容以高级排版模板呈现
- free 模板保留但不作为主线

### 2.3 右侧 Inspector

三个 tab，切换显示：

**内容 tab**：
- 根据当前页模板动态生成结构化表单
- CODE 模板显示：eyebrow / title / description / fileName / language / code
- FLOW 模板显示：eyebrow / title / steps / footer
- STAT 模板显示：title / stats 列表
- QUOTE 模板显示：quote / author
- STATEMENT 模板显示：eyebrow / statement / subtext
- 旧模板 fallback：通用元素列表

**模板 tab**：
- 模板卡片网格显示
- 每张卡片：预览缩略图、模板名称、类型标签
- 点击切换模板
- 当前模板高亮

**设置 tab**：
- 演示标题
- 快捷键说明
- 导出 JSON
- 重置示例数据

### 2.4 顶部栏

- 左侧：项目名称（可编辑 inline）
- 中间：页码 / 比例 / 分辨率信息
- 右侧按钮组：
  - 编辑 / 演示切换（toggle）
  - 撤销 / 重做
  - 重置
  - 导出（下拉：JSON / HTML）
  - 播放（▶，进入全屏演示）

## 3. 视觉风格关键词

| 方向 | 描述 |
|------|------|
| 整体 | 浅色、高级留白、editorial typography |
| 卡片 | 白色、圆角 12px、细边框、柔和阴影 |
| 背景 | 浅灰/暖白色表面 + 细点阵网格 |
| 配色 | 黑白灰为主，少量 indigo 强调色 |
| 字体 | Inter + Noto Sans SC，大字重排版 |
| 图标 | Lucide 细线风格 |
| 间距 | 丰富留白，不拥挤 |
| 动画 | 克制使用，更多用于演示模式 |
| 避免 | Office 风格复杂工具栏、传统 PPT 工具感、密集 icon toolbar |

## 4. 产品交互原则

1. **模板驱动优先** — 用户先选模板，再填内容，不是从空白 slide 开始拖拽
2. **结构化内容编辑优先** — 右侧内容 tab 根据模板类型显示结构化表单，用户不必面对元素列表
3. **自由拖拽只是高级补充** — free 模板保留但不作为默认体验
4. **画布主要用于预览和轻量编辑** — 文字 inline 编辑可用，但画布不是主编辑工具
5. **模板库是核心体验** — 用户通过切换模板来改变布局，而非手动拖拽每个元素
6. **演示和导出体验要靠前** — 顶部栏"播放"和"导出"入口应当醒目

## 5. 当前项目与目标差距

基于 P3 完成后的项目状态：

| 维度 | 当前状态 | 目标状态 |
|------|----------|----------|
| 左侧页面栏 | 缩略图预览 + 模板名称 badge，视觉较密 | 页码 + 类型标签 + 标题摘要，更简洁现代 |
| 中间画布 | 纯色背景，无点阵网格 | 浅点阵背景 + 状态信息栏 |
| 右侧面板 | 单列：内容编辑 + 属性设置混合 | 三 tab：内容 / 模板 / 设置 |
| 模板卡片 | 弹窗展示，4 列网格 | inspector 模板 tab 中展示 |
| 内容编辑 | 通用元素字段编辑器 | 按模板结构化表单 |
| 顶部栏 | 项目名 + 图标按钮：撤销/重做/导出/导入/演示 | 增强：编辑/演示 toggle + 播放 + 导出下拉 |
| 模板类型 | 10 种（title/bullets/image-text/quote/stats/timeline/comparison/full-image/html/content/free） | 产品化分类：FLOW/STACK/CODE/STAT/QUOTE/STATEMENT/FREE |
| free 模板 | 已实现拖拽/缩放 | 保持但不作为主线 |
| 演示体验 | 全屏 + 键盘翻页 + 演讲备注 | 需保留并优化入口 |

## 6. 后续阶段路线

### P4：Showreel 风格 UI 重构

- 三栏布局重构
- 左侧页面栏视觉升级（页码 / 类型标签 / 标题摘要）
- 中央画布视觉升级（点阵网格 / 状态栏）
- 右侧 inspector 改为 内容 / 模板 / 设置 三 tab
- 顶部栏重构（编辑/演示 toggle + 播放按钮）
- **不改 store、不改模板渲染逻辑**

### P5：模板系统产品化

- FLOW / STACK / CODE / STAT / QUOTE / STATEMENT 模板
- 模板元数据 productization（templateLabel、缩略图配置）
- 模板 tab 显示模板卡片网格
- 内容 tab 按模板生成结构化表单
- **不改旧 10 个模板渲染代码**

### P6：演示与导出体验

- 优化播放入口和体验
- 导出功能入口整合
- 更新状态报告
- tag 发布
- **不做 PDF/PPTX**
